# Kitti Oppressor MK1

Oppressor Mk1 inspired motorcycle for GTA San Andreas.

## Features

- Boost
- Glide
- In-air Turning
- Dumbfire Missiles
- Heat Seeking Missiles
- INI Configuration

## Installation

Download the Mod from LibertyCity.
Requires CLEO 4 installed, but it is better to have Essential Pack Mod;
Copy the Folder "Kitti Oppressor Mk1 0.1.3" in modloader folder;
If desired, replace the BF-400 model with a more suitable one;
Start the game. The mod is activated when using the BF-400.

## Controls

Left Shift — enable acceleration.
Left Alt button — shoot from the machine gun.
Hold RMB — for locking target for rocket.
LMB — to shoot rocket.
WASD or Up and Down arrows — control during flight.
